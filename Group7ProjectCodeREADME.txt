In this ReadMe file it will give a very simple brief explanation how to run the code/menu of the group 7 project code.

Ensure that you have the groups_code.h file in the header which contains the algorthims to run with Group7ProjectCode.cpp

Once the execution is running a menu will be prompt open which will ask to input a number between 1 to 11.

The menu will show which number will execute what algorthim.

To exit the program input 11.
